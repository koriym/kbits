# 大きな変化が起きている

**公開日:** 2026年2月

## 要約

AI業界の内部にいる創業者が、技術に詳しくない家族・友人に向けて書いた警告文。「礼儀正しいバージョンを伝えてきたが、もう現実と乖離しすぎた」という動機で書かれた、珍しい内部者視点のエッセイ。

### 「もう起きた」——業界内側での体験

著者は6年間AIスタートアップを構築してきた。2026年2月5日を境に、コーディング作業が完全にAIに移行したと述べる。

- 「アプリを作れ」と英語で説明し、4時間席を外すと**完成品が待っている**
- AIはアプリを自分で起動し、ボタンをクリックし、テストし、修正まで行う
- 返ってくるのは「テストの準備ができました」——そして通常、完璧に動作する

これは予測ではなく、著者の「月曜日の実際の仕事」だという。

### 能力の急加速を示すデータ

METR（AIの能力を測定する機関）の計測：

| 時期 | AIが単独で完了できるタスクの長さ（人間の専門家換算） |
|------|----------------------------------------------|
| 1年前 | 約10分 |
| その後 | 1時間、数時間と拡大 |
| 2025年11月 | 約5時間（Claude Opus 4.5） |

この数値は**約7ヶ月ごとに倍増**しており、最近のデータでは4ヶ月まで加速している可能性がある。

### AIが自分自身を構築し始めた

著者が最も重要と考える発展：OpenAIは最新モデルの技術文書に「このモデルは自身の作成に貢献した」と明記した。Anthropic CEOのDario Amodeiは「AIが書くコードが大半を占める」と述べており、自己改善のフィードバックループが「月単位で加速している」と語る。

### 影響を受ける職種

著者が具体例として挙げるもの（「これだけではない」と明示）：

- **法律**: 契約読解・判例調査・準備書面作成でジュニアアソシエイトに匹敵
- **金融分析**: 財務モデル・投資メモ・レポート生成
- **ライティング・コンテンツ**: 人間との区別が困難なレベルに到達
- **ソフトウェアエンジニアリング**: 数十万行のコードを正確に生成
- **医療分析**: 画像診断・検査結果解析で人間に迫る領域も

### 「試したけど大したことなかった」という反応について

2023〜2024年初頭に試した人の多くは「幻覚がある」「大したことない」と感じた——それは当時正しかった。現在のモデルはその時点から「AIの歴史では古代史に相当する」ほど変化している。また、多くの人が使う無料版は有料版より1年以上遅れているとも指摘する。

### 推奨アクション

1. **有料版（Claude or ChatGPT、月$20）に加入する** — デフォルトでなく最高性能モデルを選択する
2. **検索エンジンのように使うのをやめる** — 実際の仕事を丸ごと任せてみる
3. **毎日1時間、新しいことを試す** — 6ヶ月続ければ周囲の99%より理解が深まる
4. **財務的な余裕を作る** — 想定より早く変化が来る可能性に備える

## 論評

技術系の「AIは凄い」記事は多いが、本稿が異なるのは**業界内部者が非専門家の身近な人々に向けて書いた**という立場にある。著者は「礼儀正しい版」と「本当のこと」の乖離が大きくなりすぎた、と告白する。その個人的な切迫感が文章に滲み出ており、説得力を持つ。

「2020年2月のCovid初期」との比較は的確だ。当時も「過剰反応では」と思っていた人が数週間後に全てを変えた。著者は「私たちは今、AIにおける同じフェーズにいる」と言う。

ただし、記事中に登場する一部のモデル名（GPT-5.3 Codexなど）は現時点での正確な製品ラインとは一致しない場合がある点は留意が必要だ。しかし個別の製品名よりも、「AI能力が倍増ペースで加速している」という主張の核心は、客観的な測定データ（METR）によっても裏付けられており、エッセイの本質的な価値は損なわれない。

広範な職種への影響、経済的備え、子供の教育への示唆まで論じており、AIの社会的影響を包括的に考える上での出発点として価値が高い。
