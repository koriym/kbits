# DVDを買っておくべきだった

**原題:** We Should Have Bought the DVDs  
**著者:** Veronica Walsingham  
**公開日:** 2020年1月20日  
**ソースURL:** [https://www.nytimes.com/2020/01/20/opinion/future-dvds-streaming.html](https://www.nytimes.com/2020/01/20/opinion/future-dvds-streaming.html)  
**アーカイブ日:** 2026-02-26

---

## 要約

NYT「未来からのオピニオン」シリーズの一編。2022年を舞台にした近未来フィクションとして書かれたが、その予測は現実のものとなった。

**背景と設定：**
- 2020年1月公開。シリーズ「Op-Eds From the Future」は作家・未来学者・哲学者が近未来を舞台にフィクション形式で現代の問題を照らす企画
- 主人公は架空の2022年に生きる人物で、11個のストリーミングサービスを契約している

**サブスクリプション地獄の構造：**

| サービス | 加入理由 |
|---------|---------|
| Netflix | 最初の定額動画配信 |
| Hulu | 主要プレイヤー |
| Amazon Prime | 翌日配送とセット |
| HBO | 「ゲーム・オブ・スローンズ」 |
| Showtime | Twitter流行作品 |
| Starz | Twitter流行作品 |
| Apple TV+ | 有名俳優の番組 |
| HBO Max | ジェニファー・アニストンが移籍 |
| Peacock | スティーヴ・カレルが移籍 |
| Disney+ | 気づいたら入っていた |
| AMC More / The CW Watch | 気づいたら入っていた |

月額費用の推移: **$7.99**（Netflix単体）→ **$110.32**（11サービス合計）

**核心的な比喩：**
ヘミングウェイ『日はまた昇る』から「破産はどうして起きた？ 最初はゆっくりと、そして突然に」という一節を引用し、サブスクリプションクリープを説明。一つ一つは「月$5.99程度」の小さな追加に見えても、積み重なれば家計を破壊する。

**結論：**
コンテンツは各社のプラットフォームに分散し、好きな番組を追うためだけに次々と新サービスへの加入が必要になる。最初からDVDを購入して物理的に所有しておけばよかった——という皮肉な悟り。

---

## 論評

2020年時点では「近未来フィクション」として発表されたこの記事は、6年後の2026年において予言の書となった。Netflixの独占コンテンツ拡大、他スタジオの独立配信参入、ディズニー・ワーナー・アップルの相次ぐ参戦——まさにこの記事が描いた「漸進的な崩壊」が現実のものとなっている。

「無限のコンテンツへのアクセス」という夢が、実際には「有限の資産（DVD）を所有する時代」よりも高コストな月額課金地獄に変容するという逆説は、デジタル所有権とサービス化経済全般に通じる本質的な問いを投げかける。ストリーミングに限らず、SaaSサブスクリプション、クラウドストレージ、アプリ内課金など、同じ構造は至るところに存在する。

フィクション形式で書かれた点も価値がある。過去の予言としてではなく、体験談のリアリティで読者を引き込む手法は、現代の問題を説明する文章形式として有効だ。
